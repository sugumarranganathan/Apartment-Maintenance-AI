# 🏢 Apartment Maintenance AI Using LLM

An AI-powered Apartment Maintenance Assistant that automatically analyzes residents' maintenance complaints using a Large Language Model (LLM). The system classifies the complaint, assigns a priority, recommends the appropriate technician, estimates repair time, and generates a maintenance ticket summary.

---

##  Project Overview

Managing apartment maintenance requests manually can be time-consuming and inconsistent. This application uses an LLM to intelligently understand maintenance complaints and generate structured maintenance tickets instantly.

Residents simply describe the issue in natural language, and the AI provides:

- Issue Category
- Priority Level
- Technician Assignment
- Estimated Repair Time
- Maintenance Summary

---

##  Features

